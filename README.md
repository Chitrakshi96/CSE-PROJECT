# CSE-PROJECT
An expense manager tracks spending in categories like Food, Transportation, Entertainment, Utilities, Rent, Healthcare, and Miscellaneous. It uses a Python dictionary for totals. Users input an amount, select a category, and see the running total. View full breakdowns to monitor budgets and spending patterns.
